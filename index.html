1.1- ¿Qué significa CSS?

CSS significa Hojas de estilo en cascada.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc. Los programas que se utilizan son los navegadores, como por ejemplo: Firefox, Chrome o Edge (en-US).

1.3- ¿A qué se llama regla de CSS?

Los CSS estan ordenados por reglas, que son un conjunto de propiedades y valores que se aplicarán a una o varias etiquetas según necesitemos.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Selector:
selecciona los elementos a dar estilo. Para dar estilo a un elemento diferente, solo cambia el selector.
Declaracion:
Aplica las propiedades que uno quiere dar al elemento seleccionado en el Selector.
Propiedades:
Es lo que te permite dar un estilo a un elemento HTML.
Valor de la propiedad:
las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

1.5- ¿Qué es un selector en CSS?

Un selector nos permite aplicar estilos a elementos específicos de HTML. De esta forma es posible ser preciso al momento de aplicar diseño a un sitio web.

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

El CSS tiene diferentes tipos de selectrores los cuales son:

Universal:
Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el "*".
Por etiquetas:
ESta permite aplicar estilos a todas la etiquetas de un tipo.
Descendente:
Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo.
Clases:
Son herramientas que, como su nombre lo indica, permiten seleccionar todos los elementos que tienen un mismo nombre de clase.
ID:
Los selectores de ID de CSS buscan un elemento basado en el contenido del atributo ID .

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegadore carga el HTML
  2. Combierte el el HTML en un DOM(Document Object Model) que representa el documento en la memoria del ordenador.
  3. Busca la mayor parte de los recursos vinculados al HTML
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector.
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. Se muestra el aspecto visual de la página
